URL Shortener Microservice

User stories:

  • I can pass a URL as a parameter and I will receive a shortened URL in the JSON response.
  • When I visit that shortened URL, it will redirect me to my original link.

    • Example Creation


    https://url-shortener-ms-drosi94.herokuapp.com/urls/add/https://www.google.com

    Example Navigation


    https://url-shortener-ms-drosi94.herokuapp.com/urls/4mll Directs to https://www.google.com/
